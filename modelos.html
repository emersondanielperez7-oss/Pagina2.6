Modelos de datos

Tres formas de organizar la misma información

No existe un único tipo de base de datos: cada modelo resuelve un problema distinto sobre cómo se relacionan los datos entre sí.

Relacional

Organiza la información en tablas conectadas mediante claves. Prioriza la consistencia y el orden estricto entre datos relacionados.

PostgreSQL · MySQL

Documental

Guarda cada registro como un documento flexible, sin un esquema fijo. Ideal cuando la estructura de los datos cambia con frecuencia.

MongoDB · CouchDB

De grafos

Modela los datos como nodos y conexiones. Resalta cuando lo importante no es el dato en sí, sino cómo se relaciona con otros.

Neo4j · ArangoDB
